# Дурляндия TypeScript

## Система классов
Главный класс в решении - Modifier. Это абстрактный класс, определяющий как каждая наследующая от него сущность будет изменять характеристики дурляндца.

Данный класс имеет три метода (по одному для каждого параметра), которые принимают контекст изменения (дурляндца, местность и действие) и значение изменения с прошлого модификатора. Также данный класс имеет protected поле с названием класса и Generic метод, позволяющий для определить принадлежность к тому или иному классу. 

На каждом ходу в первую очередь получаем изменения от выполняемого действия, обрабатываем эти значения в соответствии с локацией, а потом в соответствии с национальностью дурляндца.

Действия наследуют класс Modifier и возвращают соответствующие изменения к параметрам дурляндца для контекста. Четыре возможных действия объединены в Union-класс.

Дурляндцы наследуют класс Modifier и имеют параметры отвечающие за здоровье, деньги и удовлетворённость жизнью, а также массив объектов типа Step, содержащий историю дурляндца

Местности также наследуют тип Modifier и имеют параметры, отвечающие за количество слесандр, сисяндр и чучундр в локации

Поскольку методы по умолчанию Modifier возвращают результат прошлого изменения, а изменения для всех типов действий определены (а действия применяются в первую очередь), добавить новые локации или национальности очень просто, и требует изменения лишь тех методов, чью логику мы хотим поменять.
